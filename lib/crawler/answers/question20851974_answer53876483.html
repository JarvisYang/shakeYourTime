我摘取我曾经翻译编辑的《专注完美近乎苛求——Lexus》一书中的几个小片段，以及我与这本书的作者Chester Dawson的对话。Chester Dawson曾经是美国商务周刊驻东京的记者，现在他自己开咨询公司。与Chester Dawson的对话发生在10年前，里面谈及了我们对中国汽车品牌未来的看法。10年之后再回头看看当年的对话，很有趣。

以下的小片段聊的都是雷克萨斯第一代LS。这款车的诞生逼迫着梅赛德斯-奔驰被迫将他们的S级发布时间从1989年推迟到1991年。这对当时不可一世的奔驰是巨大的羞辱，因为他们知道当时奔驰S的各种技术指标赢不了LS。

故事1

开发第一代LS时，总负责人铃木一郎要求丰田的专家们将草图塑身为风阻系数低于0.30的汽车。当时还没有哪辆豪华查拥有这么低的风阻系数。宝马735i和梅赛德赛-奔驰SE的风阻系数分部为0.32和0.37。房车的平均风阻系数为0.35。而当时丰田自己运动型轿跑车Soarer和Supra的风阻系数也没有低于0.32。0.30并不是一辆豪华车所拥有的风阻系数值，强行达到这一值必然另车身变得像跑车。他们将制动系统和悬挂系统都安装在原型车上，他们希望取得真实的风阻系数值，然后再分析哪一个机械部件影响了气流的流动。3年的时间里，丰田一共做了50次独立的风洞实验。

为了能够让气流顺畅地流过车身，车身上的玻璃甚至车身的把手都尽量与车身融为一体而不暴露在外阻隔气流流动。尾窗的倾斜角度恰到好处，气流在可以顺利与车身表面分离而不会产生很多扰流。但是，可能最精妙的设计师，尾厢盖上不经意的扰流板。很多人不加提示很难意识到这个扰流板的存在。其他改善空气动力学的设计还包括前轮前边改变气流流向的轮罩、车鼻下面的气坝，甚至排气管的外形都根据空气动力学原理重新设计和布置。每一项改善都无法独自让整车的风阻系数降低很多，但是他们联合在一起，就创造了铃木一郎所期望的奇迹。雷克萨斯LS400成为第一辆风阻系数低于0.30的豪华轿车。0.29的风阻系数也是当时在美国生产的所有房车的最低值。雷克萨斯LS400的风阻系数甚至比保时捷911还低。这样一来，雷克萨斯汽车的油耗率下降了17％，而汽车的最高时速提高了7英里/小时。

雷克萨斯是一个不爱吹牛皮的品牌，品牌里工作的人都特别低调。比如至今绝大多数人都不知道LS风阻控制背后的故事。


故事2

在各地进行路试的同时，铃木一郎又开启了“防老化”计划。丰田将雷克萨斯LS400开到各种极端天气情况下进行耐久度的测试。丰田的工程师要确信他们的车在开了5万英里（8万公里）之后车身不会出现锈蚀、严重磨损或者在操控感觉上有所改变。“防老化”计划一共有96项测试项目。雷克萨斯LS400被运到美国西南部亚利桑那州的沼泽地，丰田在那里测试新型的雷克萨斯复合型不锈钢车身板件。丰田发现普通的钢材在沼泽地区很快就会发生锈蚀而2个月后钢材变得面目全非。但是雷克萨斯LS400的车身材料在这样严酷的天气条件下暴露3个月也没有任何变化。另外一个极端测试是在澳大利亚和沙特阿拉伯的沙漠中进行的，测试项目是风挡上的隔热玻璃。


故事3

究竟什么样的车才可以一个年纪在43岁年收入10美元的打算购买梅赛德赛-奔驰车的人临时变卦来买日本豪华车？项目负责人神保正治征召了丰田60名设计师和造型师参与雷克萨斯的设计。一共有24个工程设计团队参与雷克萨斯的设计，总计1400名工程师、2300名技师和220名工人先后加入了进来。如此多的人力让世界所有的汽车公司都震惊，这相当于美国波音公司在90年代初设计波音777客机研发人员的一半。丰田汽车公司买来宝马和梅赛德赛-奔驰汽车，他们将这些汽车从一辆整车拆成零件来分析他们师怎么装配到一起，以及仪表板、座椅、中控台都是什么材料制成的。50年前，丰田也是采用这种原始的办法来学习美国和欧洲汽车的。在很多人看来这很幼稚，但是它却行之有效。丰田汽车公司从宝马和梅赛德赛-奔驰的零件上分析出德国人的质量标准。为了打赢这场对德战争，丰田设定了可以击败德国人的最低质量标准。他们将这一标准发给零配件供应商，要求他们制造出比德国人更耐用、成本更低的零件。

1989年，雷克萨斯LS400搞出巨牛逼的“香槟塔”广告（雷克萨斯在LS400的引擎盖上放了15个装满水的高脚杯，然后在马力机上，将车速提到232公里每小时，此时高脚杯里的水却没有溢出来）。广告公映之后，美国媒体不信。结果雷克萨斯邀请美国媒体，现场重复了一次实验。


以下是我与Chester Dawson的对话

这位大神长这个样子

Chester Dawson先生，在你调研丰田和雷克萨斯品牌时让你感受最深的是什么？


C D：我想是对mono-zukuri（造物，产品创新）的投入。很多美国公司将生产看作达到目的的手段，直白的表述就是，就是追求利润。但是对于日本公司而言，生产是实现自身存在价值的手段。当然，丰田显示了与其他日本公司不同的地方就是他能够持续地保持高利润，同时他的产品品质也达到全球的顶级水平。

在阅读和翻译本书时，我们有一种强烈的感觉，雷克萨斯是一个奇迹。你认为有人会拷贝走这个奇迹或者他仅仅是属于丰田汽车公司？

C D：不，我不认为丰田汽车公司的经验是不可复制的。这也是为什么我写此书的原因之一，因为我希望提炼出可以让其他企业受益的经验。当然如果有一天丰田放弃了他今天所追寻的目标——让消费者付出的每一分钱都有价值，那么他也会失去在行业的领先优势。雷克萨斯的奇迹是丰田汽车公司创造的，但是它却可以属于每个有野心的公司。

可以谈谈丰田的Scion吗？你认为丰田的这个品牌会成为雷克萨斯二世吗？

C D：Scion与雷克萨斯有些相似，但是他们在本质上有很多不同。例如，Scion品牌汽车在美国通过丰田的经销商体系来销售。而雷克萨斯在美国则使用与丰田汽车完全独立的销售网络来售车。雷克萨斯上市时针对的是业已成熟的市场——高端消费市场。因此，丰田在雷克萨斯的投入上无比巨大。但是Scion则不同，他面对的是新兴市场——先锋市场（年轻一代，我们称作Y一代人群）。树立Scion的品牌投入则要很多，是丰田首先发现并细分这一市场的。

丰田有两个主要的品牌——丰田、雷克萨斯（Scion、大发和日野可以不作考虑），而福特、大众则拥有相当多的汽车品牌，但是在市场份额上他们却输给了日本对手。你是否能够解释，为何那些老牌的欧洲和美国汽车公司没有丰田这么强大的品牌效率？

C D：我想美国汽车制造商已经意识到他们拥有太多汽车品牌是一个大问题——尤其是当他们无法让两个品牌的产品看起来不一样时。这也是为什么最近通用汽车公司停产了Oldsmobile品牌，而福特汽车公司也将尘封一些老品牌。当然，在日本也有一些有趣的事情。丰田也存在类似的“臃肿”问题。丰田在本土拥有太多销售网络，这些销售网络之间却是完全独立的，这让产品从生产基地到消费者手中的过程变得缺乏销量。现在丰田也在开始梳理他的经销商网络。

日本人和美国人在看待“品牌”时有何不同？

C D：我认为美国消费者在购买产品时更看重它是不是货真价实，品牌是第二位考虑的。而日本消费者则更看重品牌，尤其在挑选高档品牌时，价格对他们的敏感度相对较小。

你认为雷克萨斯品牌会成为欧洲市场的强力战士吗?

C D： 丰田汽车公司正努力让雷克萨斯在欧洲变得更有竞争力。他们通过更多的型号来吸引欧洲的消费者，例如柴油发动机的雷克萨斯，以及更多不同排量等级发动机的雷克萨斯。但是，丰田的雷克萨斯在欧洲还有很多事情要做。欧洲人更看重一个品牌的历史和传承，而美国消费者这方面的意识则相对较弱。

你在日本的生活如何，你在那里是一位很有名气的美国记者吗？

C D： 尽管我经常出现在日本的电视节目上，但是我想还是不要把我当成名人。我尽可能保持低调。一位好记者应该更注重倾听而不是向别人滔滔不绝。

你有自己的车吗？是一辆丰田吗？

C D：当我在东京和纽约时，我都没有拥有过汽车。不过我试驾过很多尚处于测试阶段的样车，这是真的。

很多很多中国读者乐于与你分享这本书背后的故事，这工作很难吗？

C D：我曾经低估了完成这本书所要花费的时间精力，因为在写这本书时我同时还要兼顾我的正常工作。我是用我的假期写完这本书。每个工作日我都忙碌到深夜，每个周末我都在与雷克萨斯为伴。这样的生活持续了52个星期。这快让我疯掉了，当然我身边的人也不能幸免。但是我非常高兴我写完了这本书。

现在很多中国的本土品牌也在跃跃欲试，他们中的一两个或许有一天可以成为下一个雷克萨斯或丰田。你是否有兴趣以西方专业记者的身份报道和记述他们的故事。

C D：我相信中将要成为——无论他是否准备好了——全球经济的重要中心。他已经显示了强大的市场购买力，这让包括国际汽车制造商在内的很多国际投资者震惊。我不会惊讶下一个十年中国会出现一个可以参与国际竞争的汽车品牌。日本品牌用了30年，韩国品牌用了20年成为舞台的主角。我想从现在开始，用10年的时间，中国会缔造一个丰田或现代一样的品牌。包括我在内，很多人都是这样预测。和很多汽车记者一样，我非常期望我能够见证中国的崛起。我相信很多其他汽车记者和我拥有相同的祝愿！

谢谢！
